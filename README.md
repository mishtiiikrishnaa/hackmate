# HackMate

HackMate is a college-network hackathon platform for finding teammates,
discovering events, and forming teams without the chaos of group chats
and cold DMs.

Signup is gated by verified college email domains, so every profile on
the platform belongs to an actual student at an actual college — not a
random LinkedIn ghost claiming to know React. HackMate handles
skill-based team matching with match percentages, hackathon discovery
and filtering, team dashboards, real-time chat, notifications, college
community pages with leaderboards, and a full admin panel for managing
colleges, users, and events — all wrapped in a glassmorphic, dark/light,
fully responsive UI.

**Status:** live / feature-complete.
